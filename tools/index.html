Github
Let's build from here
一个在线软件源代码托管服务平台，什么地位什么身份，不必多说，懂的都懂
详情
Gitee
DevOps 一站式研发效能平台
号称国内的 Github，但是。。。唉，懂的都懂，图片审核太离谱，一股子商业气息，但是国内速度比 Github 快太多了，也是一个选择
详情
Vercel
Build and deploy the best Web experiences with The Frontend Cloud
vercel 类似于 github pages，可以托管静态页面，但远比 github pages 强大，速度也快得多，具备 Serverless 能力，支持 github 自动部署
详情
TinyPNG
Smart WebP，PNG and JPEG compression
一个近乎无损压缩图片的网站，非会员一次性可同时上传 20 张，很好用
详情
obsidian
Obsidian is the private and flexible writing app that adapts to the way you think
一款特别好用的 markdown 笔记软件，个人用户免费，笔记里的 pycharm，插件众多，配合 git 可以三端同步，本人自用，强力推荐
详情
PicGo
图片上传、管理新体验
一款免费的自动将图片上传到图床的工具，十分方便，狠狠的推荐，可以配合 obsidian 或 typora 使用，本人自用
详情
SM.MS
Simple Free Image Hosting
一款图床平台，速度还行，但有时候不太稳定，免费 5G 空间，一次性最多上传 10 张，每天限制 200 张，个人来说够用了，建议账号登录，方便管理
详情
腾讯云 COS 对象存储
由腾讯云推出的无目录层次结构、无数据格式限制，可容纳海量数据且支持 HTTP/HTTPS 协议访问的分布式存储服务
因为 SM.MS 的速度不稳定，有时候加载图片太慢，改用了腾讯云 COS，按量付费，个人使用价格还算好，并且速度稳定，大厂加持，功能强大，推荐使用这个
详情
WallpaperHub
Free wallpapers for your PC，phone and tablet
免费壁纸网站，本站文章图片的来源，支持一波，在此致谢
详情
Pexels
才华横溢的创作者分享的免费照片和视频
免费高清壁纸，风格多样
详情
Font Awesome
Take the hassle out of icons in your website.
网页图标网站，也是本站图标的来源，支持一波，在此致谢

与 Font Awesome 类似，都是图标网站，但是比 Font Awesome 更加丰富，彩色图标比较多

网站信息查询
